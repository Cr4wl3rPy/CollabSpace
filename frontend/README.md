# Frontend - CollabSpace

Cliente React con canvas colaborativo y chat en tiempo real.

## Stack
- React 18
- Vite
- Socket.io-client
- Fabric.js
- Tailwind CSS

## Setup (Próximamente)
```bash
cd frontend
npm install
npm run dev
```
