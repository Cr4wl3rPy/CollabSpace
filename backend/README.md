# Backend - CollabSpace

Servidor Flask con WebSockets para colaboración en tiempo real.

## Stack
- Python 3.10+
- Flask
- Flask-SocketIO
- PostgreSQL

## Setup (Próximamente)
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
